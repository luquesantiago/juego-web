# Juego web — recolector de monedas

Minijuego en JavaScript puro: movés al jugador con las flechas del teclado, juntás monedas y esquivás al enemigo.

**▶ Jugalo acá: https://luquesantiago.github.io/juego-web/**

## Características

- Movimiento con flechas del teclado, con límites del tablero
- Contador de monedas recolectadas
- Enemigo que hay que esquivar
- Puntuaciones y botón de reinicio

## Tecnologías

JavaScript (vanilla), HTML, CSS. Manipulación del DOM y game loop con intervalos. Sin dependencias.

## Correr localmente

Abrí `index.html` en el navegador.
